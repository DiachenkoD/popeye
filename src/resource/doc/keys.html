special keys for the entry editor

key
 effect
Enter
 save the current entry and jump to the next language
Shift + Enter save the current entry
Tab
 save the current entry and jump to the next language
Down
 next entry
Up
 previous entry
Page Down
 next incomplete entry (only the current language)
Page Up
 previous incomplete entry (only the current language)
Control + Insert
 copy selection
Shift + Insert
 paste from clipboard
